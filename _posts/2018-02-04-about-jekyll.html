In layman's terms, Jekyll is a computer program written in Ruby, which is run from the terminal. It allows a user to generate and host a temporary site on their computer and regenerate the display as its source code is being modified. It was essential to helping this site get off the ground!

A great benefit of this program is that the site can be updated with ease. New files and media can reach you, the readers, within mere minutes the moment they are created. All it takes are some tricks and quick keystrokes.

Another one of the benefits of Jekyll is how it can auto-convert blog posts from Blogger to be uploaded into our site. Read the instructions under the "How to convert..." blog post to learn more.

There are many other tools which helped make this site happen. Keep reading for more updates and new information!
