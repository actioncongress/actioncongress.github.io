Git works as a version control system for multiple users across different devices. Imagine if several people were working on a project together, they would want to keep track of who contributed what, what the previous states were, and the current state of the project. For Git, many of those projects are applications or websites. Users can keep track from their own devices of which files to work on, and see any history, changes, or branches of that project.

GitHub takes all of the capabilities of Git, and connects them to the web. On GitHub, contributors can access with ease project files, view them with leisure in their own browser, and download versions from the web either to try out software or add to it themselves. You can even view the source code for this site at our GitHub repository here! For curious members, please contact Lauren about site content and changes.

GitHub Pages allows for free site hosting. It is a service provided by GitHub, so that no extra fee is required to publish our website to the world wide web. (The domain name is a different story!)

I hope these last posts help our members understand how the inner workings of the website run and operate smoothly.
